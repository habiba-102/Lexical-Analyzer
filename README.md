# Lexical Analyzer

A C++ implementation of a lexical analyzer (tokenizer) that processes source code input and classifies tokens based on compiler design principles.

## What it does

- Scans source code character by character
- Identifies and classifies tokens: **keywords**, **identifiers**, **operators**, **literals**, and **symbols**
- Outputs a token stream that can be used as input to a parser

## Built With

- C++
- Visual Studio

## How to Run

1. Clone the repository
   ```
   git clone https://github.com/habiba-102/Lexical-Analyzer.git
   ```
2. Open the solution file `Lexical Analyzer.sln` in Visual Studio
3. Build and run the project
4. Enter source code input when prompted

## Concepts Applied

- Finite automata and state machines
- Token classification
- String processing and character scanning
- Compiler design fundamentals
